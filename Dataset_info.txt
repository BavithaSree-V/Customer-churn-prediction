Dataset Name:
Telco Customer Churn Dataset

Dataset Description:
This dataset contains information about customers of a telecom company. 
It is used to predict whether a customer is likely to churn (leave the service) based on their personal details, subscription, and usage patterns.

Number of Samples:
~7,000 customers

Features / Columns:
- customerID: Unique ID for each customer
- gender: Male / Female
- SeniorCitizen: 1 if customer is 65 or older, 0 otherwise
- Partner: Yes / No
- Dependents: Yes / No
- tenure: Number of months the customer has stayed with the company
- PhoneService: Yes / No
- MultipleLines: Yes / No / No phone service
- InternetService: DSL / Fiber optic / No
- OnlineSecurity: Yes / No / No internet service
- OnlineBackup: Yes / No / No internet service
- DeviceProtection: Yes / No / No internet service
- TechSupport: Yes / No / No internet service
- StreamingTV: Yes / No / No internet service
- StreamingMovies: Yes / No / No internet service
- Contract: Month-to-month / One year / Two year
- PaperlessBilling: Yes / No
- PaymentMethod: Electronic check / Mailed check / Bank transfer / Credit card
- MonthlyCharges: Monthly billing amount
- TotalCharges: Total billed amount
- Churn: Yes / No (target variable)

Source:
Kaggle: https://www.kaggle.com/blastchar/telco-customer-churn

Notes:
- Some preprocessing may be done such as handling missing values, 
  encoding categorical variables, and scaling numeric features.
- Target variable is 'Churn' for prediction using ML models.
